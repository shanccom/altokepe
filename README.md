# PFOYECTO ALTOKE PE
## Descripción:
Sistema Altokepe es una plataforma inteligente para gestionar pedidos en restaurantes mediante comunicación TCP con mensajes JSON. Conecta al recepcionista, las estaciones de cocina y el panel de ranking, coordinando pedidos y notificaciones en tiempo real. Su objetivo es optimizar el flujo de trabajo y la atención al cliente de forma rápida, sincronizada y automatizada.
